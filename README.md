# tennis

Tennis but with janky physics. Made using Three.js.

## Demo:
https://leslieyip02.github.io/tennis/

## Controls:
- R            - Reset

**Player 1:**
- W            - Up
- A            - Left
- S            - Down
- D            - Right
- Spacebar     - Swing / Charge

**Player 2:**
- Up arrow     - Up
- Left arrow   - Left
- Down arrow   - Down
- Right arrow  - Right
- Enter        - Swing / Charge
